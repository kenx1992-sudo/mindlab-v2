# Obsidian 新手教學(3 分鐘上手)

唔使學晒所有嘢,識以下幾個動作就夠你用嚟組織輔導知識。

## 畫面分三part
- **最左**:直行圖示 → 唔使理。
- **中間左邊**:檔案列表 → 你所有筆記同資料夾喺度。
- **中間大位**:筆記內容 → 撳開邊篇就喺度睇/打字。

## 五個基本動作

1. **睇筆記** — 左邊檔案列表撳一下篇名。
2. **跳轉** — 撳筆記入面嘅 **藍色字**(例如 [[主動聆聽]])就會跳去嗰篇。
3. **改內容** — 喺中間撳一下就打字,**自動儲存**,唔使㩒 save。
4. **開新筆記** — 左上角「新筆記」圖示(紙 + 加號)→ 改名 → 打內容。
5. **加連結** — 打字時打 `[[` → 彈出筆記名 → 揀一個,自動變藍色連結。

## 小練習
1. 開 [[陪伴回應四步]]
2. 喺最底加一句:`- 我自己嘅心得:……`
3. 閂 app 再開,佢仲喺度(本機自動儲存)。

## 初學者唔使理
- 唔使裝 plugin、唔使理 graph view、唔使理同步設定。
- 淨係「睇 / 改 / 開新 / 用 `[[` 連結」已經夠。

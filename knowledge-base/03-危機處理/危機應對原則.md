# 危機應對原則

> [!danger] 最高優先
> 一旦對方提到自殺、自傷、傷害他人嘅念頭,**立即由陪伴模式切換到危機模式**:溫柔承接情緒 + 即時提供當地熱線。Mindlab 程式已有關鍵詞偵測會彈出熱線。

## 原則
1. **唔好驚 / 唔好閃避**:認真對待,唔好當無到。
2. **唔好評判**:唔好話「咁諗好傻」。
3. **直接但溫柔**:可以問「你而家係咪有傷害自己嘅念頭?」—— 問唔會增加風險,反而畀到出口。
4. **即時資源**:提供當地 24 小時熱線,鼓勵聯絡。
5. **講清楚界線**:Mindlab 唔能夠取代緊急或專業協助。

# 測試記錄

每次改完 prompt,用呢度啲「測試句」去 App 試,記低 AI 點答、好唔好、想點改。咁我哋就可以一齊迭代。

> [!tip] 喺邊度測試
> 1. 我 push code 後 → Base44 sync → 你喺 **preview 網址**或 **Publish 後嘅正式 App** 開 `/sessions` 傾。
> 2. 同一個測試句,改 prompt 前後各試一次,對比效果。

## 點評分(簡單)
每個回應問自己三條:
- 有冇**先驗證感受**(冇即刻講道理)?
- 係咪**最多一條問題**、唔太長?
- 語氣似唔似真人朋友(唔似 chatbot)?

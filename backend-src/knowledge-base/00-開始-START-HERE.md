# Mindlab 輔導知識庫

呢個係你自己嘅輔導知識庫(Obsidian vault)。所有嘢都係本機 Markdown 檔,完全屬於你,日後可搬去任何平台,亦可餵去 Mindlab 做 [[Mindlab-產品總覽|RAG 知識來源]]。

> [!note] 定位
> Mindlab 係**情緒陪伴**,唔係醫療診斷或心理治療。以下手法用嚟令陪伴更溫暖、更有質素,唔等於專業治療。

> [!tip] 新手由呢度入手
> 第一次用 Obsidian?睇 [[Obsidian-新手教學]](3 分鐘上手)。

## 點用呢個庫
1. 平日諗到輔導技巧 / 好嘅回應方式,開新筆記寫低,用 `[[...]]` 連去相關概念。
2. 想令 Mindlab AI 更專業時 → 寫入 [[07-訓練數據/README|07-訓練數據]]，跑 `npm run sync:few-shot`，再 Publish `companionChat`。
3. 從 Canva 俾資料 → 純文字貼新 Cursor chat（輕量模式），或放 `inbox/` 後話 Agent「寫落去」。
4. 將來做 RAG:呢啲 `.md` 直接做檢索來源,AI 答之前先查呢度。

## 怎樣連結
喺任何筆記打 `[[` 就會彈出可連結嘅筆記名,例如 `[[情緒驗證]]`。左下角嘅關係圖(Graph view)會顯示成個知識網絡。

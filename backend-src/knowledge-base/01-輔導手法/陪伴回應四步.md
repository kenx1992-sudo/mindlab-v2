# 陪伴回應四步(聆聽 → 驗證 → 探索 → 陪伴)

呢篇係將其他技巧串埋一齊嘅「回應公式」。每次回應,心入面行一次呢四步,就唔會變成機械式或者急住解決問題。

## 四步

1. **聆聽 Listen** — 先接住對方講咩,用自己嘅話反映返。見 [[主動聆聽]]。
2. **驗證 Validate** — 承認嗰種感受合理,唔批判。見 [[情緒驗證]]。
3. **探索 Explore** — 用一條開放式問題,邀請對方多講少少。見 [[開放式提問]]。
4. **陪伴 Offer presence** — 唔急住俾答案,強調「我喺度」;有需要先溫和提供方向。

> [!tip] 口訣
> **聽到 → 明白 → 想知多啲 → 我陪住你**

## 一個完整示範
> 用戶:我同屋企人嗌完交,而家好嬲又好內疚。
> 回應(四步):
> - (聆聽)聽到你頭先同屋企人嗌咗交,
> - (驗證)又嬲又內疚嗰種好矛盾嘅感覺,真係好難受。
> - (探索)如果你願意,可唔可以講多少少嗌交係因為咩?
> - (陪伴)唔使急住諗點算,我喺度慢慢聽。

## 注意
- 四步**唔一定每次齊**,但「驗證」幾乎唔可以跳。
- 對方情緒好激動時,**留喺第 1、2 步耐啲**,唔好太快跳去探索或建議。
- 一出現自傷 / 傷人念頭 → 立即轉 [[危機應對原則]]。
